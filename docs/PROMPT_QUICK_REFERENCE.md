# Prompt快速指南

**30秒学会如何向AI Coach提问**

---

## ✅ 好的Prompt = 情境 + 挑战 + 需求

### 公式：
```
我在[情境]中遇到了[挑战]，
我[背景/尝试过的事]，
能帮我[具体需求]吗？
```

---

## 5个立即可用的模板

### 1️⃣ 职业决策
```
我在考虑[选项A vs B]，
A的好处是[...]，坏处是[...]，
B的好处是[...]，坏处是[...]，
我最担心[...]。
```

**示例：**
```
"我在考虑留在大公司还是去创业公司，
大公司稳定但无聊，创业公司刺激但有风险，
我担心30岁了去创业太晚。能帮我分析吗？"
```

---

### 2️⃣ 领导力挑战
```
我作为[角色]遇到[问题]，
我尝试过[...]，
但[结果]。
我需要[具体帮助]。
```

**示例：**
```
"我作为新经理遇到授权困难，
我尝试过列出任务清单，
但总忍不住自己插手。
给我一些具体方法。"
```

---

### 3️⃣ 自我怀疑
```
我在[情况]中感觉[负面想法]，
我告诉自己[内心声音]，
我不知道[困惑]。
```

**示例：**
```
"我刚升职，但感觉不够格，
我告诉自己'他们早晚会发现我不行'，
我不知道这是正常的吗还是我真的不行。"
```

---

### 4️⃣ 技能提升
```
我想提升[技能]，
因为[原因]，
目前的障碍是[...]。
```

**示例：**
```
"我想提升演讲能力，
因为下月要给CEO汇报，
我的障碍是太紧张会忘词。"
```

---

### 5️⃣ 关系问题
```
我和[谁]在[什么方面]有问题，
具体是[...]，
我希望能[期望]。
```

**示例：**
```
"我和老板在沟通上有问题，
具体是他总是micromanage，
我希望能建立更多信任。"
```

---

## ❌ 避免 vs ✅ 改进

### 太宽泛
```
❌ "怎么当好领导？"
✅ "我刚当经理，团队里有人不服我，怎么建立权威？"
```

### 只有问题
```
❌ "我团队有问题。"
✅ "我的团队士气低落，2个人想离职，我尝试过团建但没用。"
```

### 不给背景
```
❌ "我应该跳槽吗？"
✅ "我在当前公司3年了，学到的东西越来越少，
   有个新offer薪水高30%但压力也大，我很犹豫。"
```

## 🎯 10个热门话题直接复制

### 1. 升职困惑
```
"我刚升职当[职位]，但发现[具体困难]，
之前和我[关系]的人现在[变化]，
我不知道怎么[具体问题]。"
```

### 2. 薪资谈判
```
"我下周要谈薪资，我的情况是[背景]，
市场行情是[数字]，我想要求[数字]，
但我担心[担忧]。怎么准备？"
```

### 3. 授权困难
```
"我很难把任务交给团队，总担心[担忧]，
我尝试过[方法]，但[结果]。
给我一些具体的授权方法。"
```

### 4. 反馈恐惧
```
"我需要给[谁]反馈，关于[具体问题]，
我担心[后果]，不知道怎么说比较好。"
```

### 5. 冒名顶替综合症
```
"我觉得自己[负面想法]，
即使[有成绩的证据]，
我还是觉得[内心声音]。
这正常吗？"
```

### 6. 会议发言
```
"我在[什么会议]中不敢发言，
我担心[恐惧]，
我想改变因为[原因]。
怎么克服？"
```

### 7. 时间管理
```
"我每天都很忙，但[问题描述]，
我尝试过[方法]，
但[结果]。
怎么真正提升效率？"
```

### 8. 团队冲突
```
"我的团队里[具体冲突]，
影响了[后果]，
我尝试过[介入方式]，
但[结果]。现在怎么办？"
```

### 9. 转型焦虑
```
"我想从[当前角色]转[新角色]，
我的担忧是[...]，
我有的优势是[...]，
怎么准备这个转型？"
```

### 10. 工作生活平衡
```
"我工作太忙导致[问题]，
我尝试过[方法]，
但[结果]。
怎么找到平衡？"
```

## 💡 3个深度对话技巧

### 1. 提供具体细节
```
Coach: "什么让你担心？"
你: "我担心说错话，具体是担心数据记错被CEO质疑，
    显得不专业。上次就有个同事被问住了很尴尬。"
```

### 2. 分享思考过程
```
"我在A和B之间纠结，A的好处是...但缺点是...，
B的好处是...但缺点是...，
我倾向于A，但害怕...，你觉得呢？"
```

### 3. 报告进展
```
"上次你建议我[...]，我试了，结果是[...]，
虽然[困难]，但我发现[新洞察]，
现在我有新问题[...]。"
```

## 🚀 立即开始

### 最简单的开始：
```
"我最近遇到一个挑战：[一句话描述]，
能帮我聊聊吗？"
```

### 如果不确定：
```
"我想[目标]，但不知道从哪里开始，
你能问我一些问题帮我理清思路吗？"
```

---

## 记住

✅ **越具体越好**
✅ **诚实分享感受**
✅ **愿意深入探索**
✅ **尝试并反馈结果**

❌ 不要太笼统
❌ 不要隐藏真实想法
❌ 不要急于求成
