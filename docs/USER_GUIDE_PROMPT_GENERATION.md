## 什么是AI Coaching

**Coaching vs 建议：**
- ❌ 建议："你应该这样做..."（告诉你答案）
- ✅ Coaching："你觉得什么选择最适合你？"（帮你找到自己的答案）

**AI Coach的角色：**
- 帮你理清思路
- 提出强有力的问题
- 挑战你的假设
- 支持你的成长
- 陪你探索解决方案

**AI Coach不是：**
- ❌ 不会替你做决定
- ❌ 不会给你"正确答案"
- ❌ 不是心理咨询师（如有心理问题请寻求专业帮助）

## 如何开始对话

### 方式1：直接描述你的情况

**✅ 好的开始：**
```
"我最近升职当了经理，但发现自己很难授权，
总是想把所有事情都自己做完。我该怎么办？"
```

**为什么有效：**
- 提供了背景（新经理）
- 描述了具体问题（授权困难）
- 表达了真实感受

**❌ 不太好的开始：**
```
"如何当好经理？"
```

**为什么不好：**
- 太宽泛
- 没有个人背景
- 容易得到通用建议

---

### 方式2：提出一个具体挑战

**✅ 好的提问：**
```
"我需要在下周给一个表现不佳的团队成员反馈，
但我很担心会伤害关系。我应该怎么说？"
```

**为什么有效：**
- 具体情境（下周要给反馈）
- 明确的担忧（伤害关系）
- 实际需求（怎么说）

---

### 方式3：分享你的困惑或冲突

**✅ 好的分享：**
```
"我有两个职业选择：一个是留在现在的公司当经理，
另一个是去初创公司做IC（独立贡献者）。
一个稳定但无聊，一个刺激但有风险。我不知道怎么选。"
```

**为什么有效：**
- 描述了真实的两难选择
- 列出了优缺点
- 表达了困惑

## 5种有效的提问方式

### 1. 情境+挑战型

**模板：**
```
"我在[情境]中遇到了[挑战]，
我尝试过[你做过的事]，但[结果]。
你觉得我应该[具体问题]？"
```

**示例：**
```
"我在带领一个跨部门项目中遇到了团队成员不配合的问题，
我尝试过单独谈话和设置明确目标，但他们还是经常错过截止日期。
你觉得我应该怎么处理这种情况？"
```

---

### 2. 决策困难型

**模板：**
```
"我在考虑[决策]，
选项A有[优点]但也有[缺点]，
选项B有[优点]但也有[缺点]。
我担心[你的担忧]。"
```

**示例：**
```
"我在考虑是否要接受一个海外工作机会，
它能加速我的职业发展，但意味着要离开家人和朋友。
我担心错过了这个机会会后悔，但也担心适应不了新环境。"
```

---

### 3. 自我怀疑型

**模板：**
```
"我[具体事件/情况]，
但我感觉[你的负面想法]。
我不知道[你的困惑]。"
```

**示例：**
```
"我刚被提升为总监，但我感觉自己不够格，
总是担心别人会发现我不够好。
我不知道这种感觉正常吗，还是我真的不行？"
```

---

### 4. 技能提升型

**模板：**
```
"我想提升[技能]，
目前我的水平是[现状]，
我的挑战是[具体障碍]。
你能帮我[具体需求]吗？"
```

**示例：**
```
"我想提升在高管会议中的发言能力，
目前我在这些会议上很安静，不太敢发言，
我的挑战是担心说错话显得不专业。
你能帮我准备下次会议的发言策略吗？"
```

---

### 5. 关系处理型

**模板：**
```
"我和[谁]的关系出现了问题，
具体是[问题描述]，
我尝试过[你的尝试]，
但[结果]。"
```

**示例：**
```
"我和我的老板关系出现了问题，
具体是他总是 micromanage 我，不信任我的判断，
我尝试过主动汇报进度和提前沟通，
但他还是每天都要检查我的工作。"
```

## 话题示例库

### 🎯 职业发展

**升职/晋升**
```
✅ "我刚升职当经理，发现之前和我平级的同事现在不听我的，怎么建立权威？"
✅ "我想在明年晋升，但不知道需要提升哪些技能。你能帮我分析吗？"
✅ "我拒绝了这次晋升机会，因为担心还没准备好，这是明智的选择吗？"
```

**转行/跳槽**
```
✅ "我在考虑从技术转产品，但我担心年龄太大（35岁），你觉得现实吗？"
✅ "我拿到了两个offer，一个是大公司稳定岗位，一个是小公司挑战岗位，怎么选？"
✅ "我想转行做coaching，但不知道从哪里开始，你能给我一些建议吗？"
```

**薪资/谈判**
```
✅ "我下周要谈薪资，不知道应该要多少，怎么准备这个对话？"
✅ "我发现新来的同事薪资比我高，我该怎么做？"
✅ "我想申请远程工作，但担心老板不同意，怎么说服他？"
```

✅ "我需要告诉一个老员工他的表现不达标，但担心伤害关系，怎么说？"
✅ "我的直接报告对反馈很防御，总是找借口，怎么处理？"

## 深化对话技巧

### 技巧1：提供更多背景

**第一轮：**
```
你："我想提升演讲能力。"
Coach："能具体说说是什么场合的演讲吗？"
```

**✅ 好的跟进：**
```
你："主要是给高管汇报项目进度，大概5-10分钟，
我担心他们问问题我答不上来，显得不专业。"
```

**为什么好：**
- 具体了场合（高管汇报）
- 具体了时长（5-10分钟）
- 说明了真实担忧

---

### 技巧2：分享你的思考过程

**第一轮：**
```
你："我在考虑要不要跳槽。"
Coach："什么让你犹豫？"
```

**✅ 好的跟进：**
```
你："留下来的好处是稳定，我了解这里的一切，
但缺点是成长空间有限。跳槽的好处是能学到新东西，
但我担心新环境可能不适合我。我倾向于跳槽，
但又害怕做出错误决定。"
```

**为什么好：**
- 列出了优缺点
- 表达了倾向
- 说明了担忧

---

### 技巧3：诚实回答教练的问题

**Coach问：**
```
"如果完全不考虑钱，你会选择什么？"
```

**❌ 不太好的回答：**
```
"我还是要考虑现实的..."
```

**✅ 好的回答：**
```
"如果完全不考虑钱，我会选择去做教育相关的产品，
因为我一直想做一些有意义的事情，不只是赚钱。
但现实中我有房贷压力，所以很矛盾。"
```

**为什么好：**
- 真实回答了假设问题
- 说明了内心的渴望
- 也承认了现实约束

---

### 技巧4：尝试coach建议的方法

**Coach建议：**
```
"你可以试试在下次会议上先准备一个问题，
然后主动提问。"
```

**✅ 好的跟进（下次对话）：**
```
你："我试了你建议的方法，在会议上提了一个问题。
虽然还是很紧张，但提问后感觉好多了。
但有个新问题：他们回答后我不知道怎么继续对话..."
```

**为什么好：**
- 你尝试了建议
- 报告了结果
- 发现了新挑战
- 准备继续深入

---

### 技巧5：深入探索，不急于求成

**❌ 太快求答案：**
```
你："怎么提升领导力？"
Coach："能说说具体想提升哪个方面吗？"
你："就整体提升吧，你有什么建议？"
```

**✅ 愿意深入探索：**
```
你："怎么提升领导力？"
Coach："能说说具体想提升哪个方面吗？"
你："我觉得主要是两个方面：
一是团队管理，我总感觉自己管得太多；
二是战略思考，我太关注细节，缺少大局观。"
```

**为什么好：**
- 愿意回答问题
- 自己先思考了
- 具体化了需求

## 常见问题

### Q1: AI Coach会给我直接的建议吗？

**A:** AI Coach主要用提问和引导的方式帮助你找到自己的答案。
有时候也会提供一些框架和工具，但最终决定权在你。

**示例：**
```
你："我该不该接受这个offer？"

Coach（不会说）： "你应该接受，因为..."
Coach（会说）： "我们来分析一下：
1. 这个offer最吸引你的是什么？
2. 最大的担忧是什么？
3. 如果不接受，6个月后你会怎么想？"
```

---

### Q2: 我应该问什么类型的问题？

**A:** 任何关于职业发展、领导力、个人成长的问题都适合。

**✅ 适合的话题：**
- 职业决策
- 领导力发展
- 团队管理
- 沟通技巧
- 自信和心态
- 时间管理
- 人际关系

**❌ 不适合的话题：**
- 心理健康问题（请寻求专业心理咨询）
- 法律问题（请咨询律师）
- 医疗问题（请咨询医生）
- 财务投资建议（请咨询财务顾问）

---

### Q3: 一次对话应该多长？

**A:** 没有固定长度，但建议：

**快速问题：** 3-5轮对话
- "下周有个困难对话，怎么准备？"
- "我应该要多少薪资？"

**深入探索：** 10-20轮对话
- "我该怎么规划职业发展？"
- "我如何从技术转管理？"

**持续成长：** 多次对话
- 每周/每两周check in
- 报告进展和新挑战
- 深入某个话题

---

### Q4: AI Coach会记住我们的对话吗？

**A:** 是的，AI Coach会在当前会话中记住对话历史，
帮助你持续深入。但建议你自己也记录重要insights。

---

### Q5: 我可以同时讨论多个话题吗？

**A:** 可以，但建议：

**✅ 好的做法：**
```
你："我最近有两个挑战：
一是团队管理问题，二是职业发展方向问题。
我们先聊聊团队管理吧..."
```

**为什么好：**
- 列出了所有话题
- 选择了优先级
- 逐个深入

**❌ 不太好的做法：**
```
你："我团队有问题，职业也迷茫，薪资也想谈，
还有演讲能力要提升，你觉得我该怎么做？"
```

**为什么不好：**
- 太多话题混在一起
- 难以深入任何一个
- 容易得到肤浅的回答

---

### Q6: 如果我觉得Coach的问题没有帮助怎么办？

**A:** 你可以：
1. **直接说出来：** "这个问题我不太确定怎么回答"
2. **要求换个角度：** "能从另一个角度问吗？"
3. **说明你的需求：** "我其实更希望得到一些具体的方法，而不是探索性问题"

**示例：**
```
Coach："你觉得为什么会这样？"

你："我其实已经分析过原因了，
我现在更想知道具体应该怎么做。
能给我一些actionable的建议吗？"

Coach："好的，那我们聚焦在行动上..."
```

## 快速开始模板

### 模板1：职业决策
```
我正在考虑[具体决策]，
目前的情况是[背景信息]，
我的选项有[选项A/B/C]，
我担心的是[你的担忧]。
能帮我理清思路吗？
```

### 模板2：领导力挑战
```
我作为[角色]遇到了[具体挑战]，
团队/情况是[背景信息]，
我尝试过[你的尝试]，
但结果是[结果]。
我需要[具体帮助]。
```

### 模板3：技能提升
```
我想提升[具体技能]，
因为我[原因]，
目前我的水平是[现状]，
我的障碍是[具体障碍]。
你能帮我制定一个计划吗？
```

### 模板4：关系问题
```
我和[谁]在[什么方面]有问题，
具体表现是[具体描述]，
这让我感觉[你的感受]，
我希望达到[期望结果]。
```

### 模板5：自我怀疑
```
我在[具体情况]中感到[负面情绪]，
我告诉自己[你的内心想法]，
我不知道[你的困惑]，
这正常吗？
```

## 总结

### 成功coaching对话的5个要素

1. **具体** - 不要太笼统
2. **真实** - 分享你的真实想法和感受
3. **开放** - 愿意回答问题和探索
4. **行动** - 尝试建议并报告结果
5. **持续** - 多次对话，持续深入

### 现在就开始

**最简单的方式：**
```
"我最近在[你的情况]中遇到了一些挑战，
能帮我聊聊吗？"
```

**记住：**
- 没有愚蠢的问题
- 越具体越好
- 诚实最重要
- 这是你的成长时间
